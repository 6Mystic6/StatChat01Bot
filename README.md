# Как запустить бота

Для запуска установите библиотеки из файла `requirements.txt` и создайте пустые папки с названиями:
```
1. AudioFile
2. Cert
3. Photo
4. PhotoOut
5. StatPlot
```

Создайте файл `config.py` и укажите ваш токен:

```Python
TOKEN = "Токен вашего бота"
```

И зпустите бота командой
```
python Main.py		# или
python3 Main.py
```
